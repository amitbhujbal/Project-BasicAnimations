# Project-BasicAnimations

A simple UIKit demo project showing common animation techniques using UIView and Core Animation.

## Demo

![Demo.](https://amitbhujbal.com/images/demo-Project-BasicAnimations.gif "Demo")

## Features

- Enlarge (Scale Up)
- Move (Translate)
- Rotate
- Shake (CABasicAnimation)
- Drop Animation
- Reset State

## Requirements

- iOS 15+
- Xcode 15+
